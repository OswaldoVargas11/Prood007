# Subprocesadores de Lawzora

Lawzora recurre a los siguientes proveedores (subencargados del tratamiento) para prestar el servicio. Esta
lista se incorpora por referencia al Acuerdo de Encargado del Tratamiento (DPA). Cualquier alta o sustitución
se notificará con una antelación mínima de 30 días a quienes estén suscritos a los avisos, con derecho a
oponerse por motivos razonables de protección de datos.

| Proveedor          | Finalidad                                                          | Ubicación        |
| ------------------ | ------------------------------------------------------------------ | ---------------- |
| Neon               | Base de datos gestionada                                           | UE (Frankfurt)   |
| Fly.io             | Alojamiento de la aplicación                                       | UE (Frankfurt)   |
| Cloudflare R2      | Almacenamiento de documentos                                       | UE / global (R2) |
| Brevo              | Envío de correo transaccional                                      | UE               |
| Anthropic          | Funciones de IA (cuando se activan)                                | EE. UU.          |
| Voyage AI          | Indexación semántica (cuando se activa)                            | EE. UU.          |
| Stripe             | Procesamiento de pagos                                             | UE / EE. UU.     |
| Google / Microsoft | Integraciones de correo/calendario (cuando el usuario las conecta) | UE / EE. UU.     |

## Cómo recibir avisos de cambios

Suscríbete a los avisos de subprocesadores desde Ajustes. Quien no esté suscrito renuncia al aviso previo
individualizado, sin perjuicio de la publicación de los cambios en esta página.
